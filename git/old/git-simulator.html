üí° Git Concepts

Git Repository Basics

A Git repository tracks changes to files over time. The basic workflow involves:

  • Working Directory: Your actual files
  • Staging Area: Files prepared for commit
  • Repository: Committed snapshots

Branching and Merging

Branches allow parallel development:

  • Branch: Independent line of development
  • Merge: Combine changes from different branches
  • Rebase: Replay commits on top of another branch

Remote Repositories

Collaborate with others using remote repositories:

  • Origin: Default remote repository
  • Fetch: Download changes without merging
  • Pull: Fetch and merge changes
  • Push: Upload your changes

Advanced Features

Advanced Git features for complex workflows:

  • Stash: Temporarily save changes
  • Tags: Mark specific commits
  • Cherry-pick: Apply specific commits
  • Interactive Rebase: Edit commit history
